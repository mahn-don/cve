# Install
- Python 3
```bash
$ pip install -r requirements.txt
```
# Usage
- Help
```bash
$ python3 cve.py -h
```
- Update data
```bash
$ python3 cve.py -u
```
- Find cve with last modified date
```bash
$ python3 cve.py -d 2021-08-30
```
- Find cve with last modified date and keyword
```bash
$ python3 cve.py -d 2021-08-30 -k linux
```
